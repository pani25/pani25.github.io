# My Portfolio Website

A pixel game-themed portfolio website showcasing my projects, certificates, and contact information.

## Features
- Pixel art design with retro gaming aesthetic
- Animated landing page with character sprites
- Smooth transitions and interactive elements
- Fully responsive design
- Project and certificate galleries
- Contact information and social media links

## Technologies
- HTML5
- CSS3 (Animations & Flexbox)
- Vanilla JavaScript
- Press Start 2P Font (Google Fonts)
